Game

Build a simple game of: ✊✋✌️

  1. Create an game.html page
  2. Create a game.js file and add it to the page
  3. Divide the page into zones we'll call input and output
  4. Add three buttons to the input zone page one each for ✊✋✌️
  5. Add code so that we react when any of those buttons is pressed.
    1. Select the buttons and bind them to a variable
    2. Loop over the array-like collection of buttons
    3. Connect each button to a function called 'play' that will run whenever the button is pressed.
    4. Create the play function
  6. Generate a random selection of ✊✋ or ✌️ for the computer
  7. Generate a statement to say who won.
  8. Invite player to play again.
